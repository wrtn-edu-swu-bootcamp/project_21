# 경쟁 인식 리프레이밍 질문 웹앱 기획안

## 📋 프로젝트 개요

### 프로젝트명
**"다시, 생각" (Re:Think)** - 경쟁 인식 리프레이밍 질문 웹앱

### 개발 목적
대한민국 청년들이 입시와 취업 과정에서 겪는 과도한 경쟁과 구조적 압박을 완화하고, 자기비난에서 벗어나 건강한 자아 인식을 형성할 수 있도록 돕는 디지털 웰빙 도구 개발

### 개발 기간
프로토타입: 4-6주 예상

### 타겟 사용자
- 입시를 준비하는 고3, 재수생
- 취업을 준비하는 대학생 및 취업준비생 (20-30대)
- 학벌/스펙 중심 사고에서 벗어나고 싶은 청년층
- 경쟁 압박으로 인한 정신적 피로를 느끼는 모든 청년

---

## 🎯 문제 정의

### 1. 핵심 문제
**과도한 경쟁에서 발생하는 자동적 자기비난과 학벌·스펙 중심 사고**

### 2. 현상 분석

#### A. 자동적 자기비난
- "나는 왜 이것밖에 안 될까?"
- "다른 사람들은 다 잘하는데..."
- 실패를 개인의 무능력으로 귀인
- 자존감 저하와 우울증으로 이어짐

#### B. 학벌·스펙 중심 사고
- 성적/학벌/자격증으로만 자신을 평가
- 외부 기준에 의존한 자기 가치 판단
- 본질적 관심사와 재능 무시
- 타인과의 끊임없는 비교

#### C. 구조적 압박
- 제한된 자원(일자리, 대학 정원)에 대한 과열 경쟁
- 사회 시스템이 만든 경쟁을 개인 책임으로 내면화
- 경쟁 구조에 대한 비판적 사고 부재

### 3. 문제의 영향
- **정신건강**: 불안, 우울, 번아웃
- **관계 악화**: 동료를 경쟁자로만 인식
- **창의성 저하**: 정해진 길만 추구
- **사회적 비용**: 청년 세대의 행복도 저하

## 💡 솔루션

### 핵심 아이디어
**일상적이고 반복적인 질문을 통해 경쟁에 대한 인식을 점진적으로 리프레이밍**

### 접근 방법

#### 1. 비강제적 개입
- 하루 1회, 부담 없는 질문 제공
- 답변 강제 없음 (선택적 참여)
- 짧은 시간 투자 (5-10분)

#### 2. 인지 재구성 (Cognitive Reframing)
- 경쟁을 개인 문제가 아닌 구조적 문제로 재인식
- 자기 가치를 외부 기준에서 내부 가치로 전환
- 비판적 사고 능력 배양

#### 3. 지속적 성찰
- 매일 같은 시간, 새로운 질문
- 과거 답변 기록으로 변화 추적
- 장기적 인식 변화 유도

## 🎨 주요 기능

### 1. 하루 1회 리프레이밍 질문

#### 질문 카테고리 (4가지)

**A. 경쟁 구조 인식**
- "이 경쟁은 누가 만든 걸까?"
- "이 기준은 언제부터 중요해졌을까?"
- "이 경쟁에서 이득을 보는 사람은 누구일까?"
- "다른 나라의 청년들은 어떻게 살고 있을까?"

**B. 자기 가치 재발견**
- "이 기준이 없었다면 나는 무엇을 하고 있을까?"
- "내가 진짜 잘하고 싶은 건 뭘까?"
- "나만의 강점은 무엇일까?"
- "10년 후 나는 무엇을 기억하고 싶을까?"

**C. 관계 재인식**
- "주변 사람들도 같은 압박을 느낄까?"
- "협력이 가능한 영역은 어디일까?"
- "내 경쟁자는 정말 다른 사람일까?"
- "함께 할 수 있는 일은 없을까?"

**D. 시스템 성찰**
- "이 시스템의 수혜자는 누구일까?"
- "이 구조는 바꿀 수 있을까?"
- "내가 할 수 있는 작은 저항은 무엇일까?"
- "더 나은 사회는 어떤 모습일까?"

#### 알림 시스템
- 첫 방문 시 알림 시간 설정 (기본: 오전 9시)
- 브라우저 알림 권한 (선택 사항)
- 앱 접속 시 "오늘의 질문" 강조 표시
- 같은 날짜는 같은 질문 유지

### 2. 답변 작성 및 저장

#### 답변 인터페이스
- **자유 텍스트 입력**: 최소/최대 길이 제한 없음
- **감정 상태 기록**: 5단계 이모티콘 선택
  - 😢 매우 힘듦 (1점)
  - 😟 힘듦 (2점)
  - 😐 보통 (3점)
  - 🙂 괜찮음 (4점)
  - 😊 좋음 (5점)
- **태그 추가**: 자유롭게 키워드 입력 (예: #입시 #취업 #학벌)
- **임시 저장**: 브라우저 종료 시에도 작성 중인 내용 보존

#### 데이터 구조
```javascript
{
  id: "uuid-v4",
  questionId: "q_001",
  question: "이 경쟁은 누가 만든 걸까?",
  answer: "사용자의 답변 텍스트...",
  mood: 3,
  tags: ["입시", "대학"],
  timestamp: "2026-01-22T09:30:00.000Z",
  edited: false
}
```

### 3. 기록 보기 (히스토리)

#### 보기 옵션
- **타임라인 뷰**: 날짜순 정렬
- **카테고리별 뷰**: 4가지 카테고리로 그룹화
- **감정별 뷰**: 감정 상태별 필터링
- **태그별 뷰**: 자주 쓴 태그로 분류

#### 기능
- **검색**: 질문/답변 내용 검색
- **수정**: 과거 답변 수정 가능
- **삭제**: 개별 답변 삭제
- **즐겨찾기**: 중요한 답변 별표 표시

### 4. 통계 및 변화 추적

#### 대시보드 지표
- **총 답변 수**: "지금까지 OO번 생각했어요"
- **연속 답변 일수**: 스트릭 표시 (🔥 7일 연속!)
- **참여율**: 월별/주별 참여도 그래프
- **감정 변화**: 감정 상태 추이 라인 차트
- **인기 주제**: 가장 많이 사용한 태그 워드클라우드
- **카테고리별 분포**: 어떤 질문에 가장 많이 답했는지

#### 인사이트 제공
- "최근 2주간 감정 점수가 평균 0.8점 상승했어요 📈"
- "경쟁 구조 인식 카테고리를 15번 탐구했어요"
- "가장 자주 생각한 주제는 '#학벌'이에요"

### 5. 답변 공유

#### 공유 방식
- **텍스트 복사**: 질문 + 답변 + 날짜 포맷으로 복사
- **이미지 생성**: 예쁜 카드 디자인으로 이미지 다운로드
- **SNS 공유**: 카카오톡, 트위터, 인스타그램 스토리

#### 익명성 보장
- 닉네임 표시/숨김 선택
- 날짜 표시/숨김 선택
- 민감한 내용 자동 감지 알림

### 6. 질문 풀 관리

#### 사용자 기능
- 전체 질문 목록 보기
- 질문 미리보기
- 특정 질문 다시 답변하기
- 커스텀 질문 추가 (향후 기능)

### 7. 설정

#### 커스터마이징
- **알림 시간 설정**: 원하는 시간대 선택
- **테마 선택**: 라이트/다크 모드
- **알림 On/Off**: 브라우저 알림 활성화/비활성화
- **데이터 관리**:
  - JSON 파일로 내보내기 (백업)
  - JSON 파일 가져오기 (복원)
  - 모든 데이터 삭제

**"압박이 아닌 위로, 경쟁이 아닌 성찰"**

## 🌟 차별화 포인트

### 1. 비판적 사고 유도
기존 웰빙 앱들이 개인의 마음가짐 변화에 초점을 둔 것과 달리, **구조적 문제를 인식**하도록 돕습니다.

### 2. 강제하지 않는 개입
매일 알림을 보내지만, 답변을 강요하지 않습니다. 사용자의 자율성을 존중합니다.

### 3. 장기적 변화 추적
단순한 일기가 아닌, **생각의 변화를 시각화**하여 성장을 확인할 수 있습니다.

### 4. 완전한 프라이버시
서버를 거치지 않아 완벽한 익명성과 안전성을 보장합니다.

### 5. 청년 세대 맞춤 디자인
MZ 세대에게 익숙한 UI/UX와 공유 기능으로 쉽게 접근 가능합니다.

## 🚧 기술적 과제 및 해결 방안

### 1. 브라우저 알림 권한
**문제**: 사용자가 알림을 거부할 수 있음  
**해결**: 인앱 알림 배너로 대체, 선택적 기능으로 제공

### 2. 데이터 손실 위험
**문제**: 브라우저 캐시 삭제 시 데이터 손실  
**해결**: 정기적으로 백업 권장 알림, 내보내기 기능 제공

### 3. 다양한 기기 동기화
**문제**: LocalStorage는 기기 간 동기화 불가  
**해결**: 프로토타입에서는 단일 기기 사용 전제, 향후 클라우드 동기화 추가

### 4. 접근성
**문제**: 다양한 사용자 환경 대응  
**해결**: ARIA 속성 추가, 키보드 네비게이션 지원, 스크린 리더 테스트
